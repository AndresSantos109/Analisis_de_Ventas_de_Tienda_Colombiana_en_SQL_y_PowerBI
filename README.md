# Analisis_de_Ventas_de_Tienda_Colombiana_en_SQL_y_PowerBI
Este proyecto tiene como objetivo analizar el comportamiento de las ventas de una empresa comercial utilizando SQL para la preparación de los datos y Power BI para la visualización.
Se desarrollaron consultas avanzadas en PostgreSQL para explorar insights, métricas clave de desempeño, limpiar y transformar la información, y posteriormente se integraron en Power BI para construir un dashboard interactivo que permita la toma de decisiones estratégicas.

# Documento que encontrará
* Archivo SQL con código empleado para crear esquemas y tablas en la base de datos
* Archivo SQL con código empleado para identificación de insights previos con diferentes "Niveles" de dificultad
* Archivo SQL con código empleado para análisis y preparación de la data previo a importación a PowerBI
* Documento Word con análisis del proyecto con insights encontrados a partir de la metodología CRISP-DM

# Tecnologías utilizadas
* PostgreSQL → Creación de vistas, consultas analíticas, detección de valores atípicos, duplicados y nulos.
* Power BI → Visualización de datos, creación de métricas DAX, KPI y paneles interactivos.
* DAX → Cálculo de KPIs como crecimiento mensual, rentabilidad y participación de clientes nuevos.

# Habilidades aplicadas
* SQL avanzado (CTEs, funciones agregadas, joins, correlaciones).
* Modelado de datos y limpieza.
* DAX y diseño de KPIs.
* Visualización e interpretación de métricas.
* 
# Principales procesos realizados
1. Limpieza y validación de datos en SQL
* Eliminación de duplicados y valores nulos.
* Identificación de valores atípicos (3 desviaciones estándar).
* Normalización de campos y actualización de información.
2. Modelado de datos
* Creación de vistas materializadas en PostgreSQL.
* Conexión con Power BI mediante el modo Import.
* Creación de una tabla calendario con DAX para análisis temporal.
3. Cálculo de KPIs y métricas clave
* Ventas totales del último mes.
* Crecimiento mensual y anual.
* Rentabilidad acumulada del año.
* Porcentaje de clientes nuevos vs recurrentes.
* Mes con mayores ventas históricas.
4.Visualización e insights
* Dashboard con barra lateral interactiva.
* Indicadores de rendimiento (KPI) con comparación mensual.
* Gráficos de tendencia, categoría y participación de clientes.

# Dashboard final
El dashboard desarrollado en Power BI permite a los directivos visualizar un panorama general de la compañía con sus principales KPIs, la evolución de ventas, rentabilidad y comportamiento de clientes de forma dinámica y actualizada. Al igual de en secciones apates informes detallados por producto, tiendas o empleados (conocer rendimiento de los mismos)
